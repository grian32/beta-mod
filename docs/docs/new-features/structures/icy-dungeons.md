# Icy Dungeons

Icy dungeons are rare underground structures that spawn in Taiga and Tundra biomes just below the surface. They include two new building blocks, Icy Stone & Icy Cobblestone along with various rewards.

![An Icy Dungeon](/img/icy_dungeon.png)

## Frost Root

Frost Root Seeds can be found among an icy dungeon's loot. They grow only in Taiga and Tundra biomes. The harvested Frost Root can be cooked to restore two hearts.

:::note
While the regular frost roots stack, the cooked ones do not.
:::
