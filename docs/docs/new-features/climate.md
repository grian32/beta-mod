# Climate

The climate feature introduces new behaviour based on the climate/biome you are in.

## Birch Trees

Birch trees now spawn with flowers around them, including a new Blue Peony flower.

![Birch Trees surround by Flowers](/img/climate_birch_tree.png)

## Saplings

All leaf types drop the same sapling. The tree type is determined when that sapling is planted:
- In Taiga and Tundra biomes it will turn into a spruce sapling. ![Spruce Saplings in a Tundra](/img/spruce_climate_saplings.png)
- Near at least three Blue Peonies it will turn into a birch sapling.
- In a Desert biome, it requires at least three nearby water blocks. Without them, the sapling will turn into a deadbush. With water, it turns into an oak sapling, or a birch sapling if enough Blue Peonies are also nearby. ![A setup for Birch Saplings in the desert](/img/desert_climate_saplings.png)
- In all other cases, it remains an oak sapling.
